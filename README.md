Simple OOP Calculator Class (C++)

A robust C++ Calculator class implementation using Object-Oriented Programming (OOP) principles to perform arithmetic operations with error handling.

🚀 Features

Basic Arithmetic: Supports Addition, Subtraction, Multiplication, and Division.

Float Support: Handles decimal numbers for precise calculations.

Error Handling: Built-in protection against "Division by Zero".

State Management: Automatically tracks and labels the last operation performed.

Clear Function: Reset all values and operations with a single call.

## 💻 Code Example
` cpp
clsCalculator calc1;
calc1.Add(12.5);
calc1.PrintResult(); `

📋 Sample Output

Result After Clear 0 is: 0
Result After Adding 12 is: 12
Result After Subtracting 2 is: 10
Result After Multiplying 3 is: 30
Result After Dividing 6 is: 5

🛠 Concepts Used

Encapsulation: Using private members to protect the calculator's state.

Methods: Implementation of class functions to modify and retrieve data.

Logic Handling: Using if-else for safety checks.
